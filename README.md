# Madeira 2026

Rejseguide-PWA til familiens Madeira-tur, 30. marts – 6. april 2026.

**Live:** [jsmariegaard.github.io/madeira-2026](https://jsmariegaard.github.io/madeira-2026/)

## Features

- Day-by-day program with expandable event details
- Interactive map with POIs, restaurants, hikes, and accommodations
- Multiple tile layers (standard, topographic, satellite)
- Offline map tile caching
- Restaurant guide with ratings and filters
- Weather links and resources
- Favorites system (yes/no/maybe) on places and activities
- Dark mode
- Installable PWA with offline support

## Tech Stack

- React 19, TypeScript, Vite 8
- Tailwind CSS v4
- Leaflet / react-leaflet
- vite-plugin-pwa (Workbox)
- GitHub Pages via GitHub Actions

## Development

```bash
npm install
npm run dev
```
